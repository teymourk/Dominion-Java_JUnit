1. Create your own variables

Creating your own varibles is a very important thing to rememeber. In this case it would be very easy to just change the information on the base URL.However, in the future if you have multiple links, it would be very useful to just have a value in a variable and use that variable to make the call. Because if you have to change something then you wouldnt have to go through every single link. You could just muniplate the variable. See example Below.

Explanation on The variables

  1. type you could either request a "GET" or "POST". In this case we are sending a "GET" Request.
  2. baseURL is essensially the API URL that contains all the datas for sound cloud.
  3. query is the Track Id that we are asking for. We are going to discuss how you could get different track ids later.
  4. apiKey is where you put your Application ID to get permission from the API to access the Data.
  5. request just sends an HTTP Request to the API.
