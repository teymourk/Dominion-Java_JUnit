Embed SoundCloud Track Widget

So far we have learned how to get track ids and use that to get the track information using the API base URL. Now lets move to the cooler stuff. In this section Im going to show you guys how to embed the track widget using the track ID we got from the last tutorial.

Using JSFiddle, im going to show you steps on how you could apply this to your own application.This could become very useful in the future when you are trying to play music or tracks on your own application.

Now just like how we applied the SDK in the last tutorial, we are going to have to do the same thing by applying this line of code. src="https://w.soundcloud.com/player/?url=http://api.soundcloud.com/tracks/181164088." dont worry too much about the frame. Its only used to create the frame for the widget.The next line of code src="https://w.soundcloud.com/player/api.js" imports the library for the widget functionality like pausing and playing.
