Making The Request

For any API, In order to get the JSON data back in an Array, You are required to make a call.
This can be done with the Client-ID you recieved when you registered an App under your account.
There are many calls that can be done to the API but for this example, we are going to use to get
information about a specific track.

SoundCloud API Refrence

By Going to the SouncCloud API Refrence, you can see that they are many other API Calls that can be used to get some data back. However, for this tutorial we will be using the Track URL which gives us all the information about a Data. You could also replace the client ID with the own ID you recieved when you registered.

http://api.soundcloud.com/tracks/13158665?client_id=5230ee8952a3bc806d3a14d7d7997df6

Now lets break down the link

  • api.soundcloud.com/ : This URL is essentialy the URL to the api which has all the libraries and tags for the datas that get requested by developers.

  • track/ : This is to specify which type of data you are looking for. by going to the API Refrence link above you can see all the different option you have. Forexample, you could look for certain playlist or users.

  • 13158665 : This number is based on every track that is on Soundcloud. Its basically our Query. Using this ID, we can get access to the data of the song we are looking for. This ID is unique to every track so you have to make sure it is correct.

  • ?client_id= :Client ID is jsut your key for the app which gives you the permission to view the document. Without this there is no way youll be able to view the JSON document.
